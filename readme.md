# Library Management System

## Overview
A complete Library Management System built with Python, Flask, MySQL and Pandas.

## Features
- REST API with Flask
- MySQL Database
- ETL Pipeline with Pandas
- Data Analysis and Visualization

## Technologies Used
- Python
- Flask
- MySQL
- SQLAlchemy
- Pandas
- Matplotlib

## How to Run
1. Install dependencies: pip install -r requirements.txt
2. Run API: python app_using_mysql.py
3. Run Analysis: python analysis.py
4. Run ETL Pipeline: python ETL-pipeline.py
